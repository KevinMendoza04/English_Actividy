# Simple Product Checker

## Description

This project is a simple Python program.  
It is a basic example of how lists and functions work together.  
The program is a console-based application that allows users to view and search for products.

## How it works

1. The program defines a list of products.
2. The function displays all products using a loop.
3. The program asks the user to enter a product name.
4. The function searches for the product by comparing names.
5. The program shows a message depending on the result.

## Status

> The program is currently running and demonstrating basic search functionality.
